Vital Statistic Recorder

This is a hobby project which began with an aim to keep up with the changing vitals of my parents. The application takes data manually and stores it in a database format.

Features of the application

The interface and design of the Vital Statistics Recorder aim to be the most simplistic. The targeted uses require very low technology skill. The layout is designed based on a calculator interface, which is a simplistic and intuitive approach for a user. The application is designed to be very lightweight and take up minimalistic space to run on a very basic mobile phones.

  • Single page app to avoid any complicated browsing.
  • Calculator layout.
  • Screen clearing button.
  • ENTER button for saving the data.
  • DATA, UP and DOWN buttons for data retrieval and scrolling the database.
  • Provides automatic timestamp while saving the data.
